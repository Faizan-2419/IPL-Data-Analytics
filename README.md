# IPL Data Analysis Project

## Overview
This project focuses on analyzing ball-by-ball data from the Indian Premier League (IPL) to extract meaningful insights about players and teams.

The analysis is performed using Python libraries such as Pandas, NumPy, Matplotlib, and Seaborn.

---

## Key Objectives
- Identify top run-scoring batsmen
- Analyze players with most 50s and 100s
- Calculate and compare strike rates
- Find top wicket-taking bowlers
- Determine most economical bowlers
- Analyze team performance (matches, wins, losses)

---

## Technologies Used
- Python
- Pandas (Data Cleaning & Manipulation)
- NumPy (Numerical Operations)
- Matplotlib (Visualization)
- Seaborn (Advanced Visualization)

---

## Data Preprocessing
- Handled missing values in key columns
- Standardized categorical data (team names)
- Applied encoding using `pd.get_dummies`
- Cleaned dismissal and extras-related fields

---

## Key Analysis Performed

### Batting Analysis
- Top 10 batsmen by total runs
- Players with most 50s and 100s
- Strike rate comparison of top players

### Bowling Analysis
- Top 10 bowlers by wickets
- Most economical bowlers based on runs conceded per over

### Team Analysis
- Total matches played by each team
- Wins and losses calculation
- Comparative visualization of team performance

---

## Visualizations
The project includes multiple visualizations:
- Bar charts (Top batters, bowlers, team wins)
- Dual comparison plots (50s vs 100s)
- Line + Bar combined chart (team performance)
- Pie chart (win distribution)
